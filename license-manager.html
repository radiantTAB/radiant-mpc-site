Radiant License Manager
Issue and track license keys for the Radiant Suite.

The License Manager is the administrator's console — used by Radiant to generate and manage license keys. Each Radiant application validates its key independently, on the user's own machine, with no internet connection.

Features

What the License Manager does.

One console for licensing the entire Radiant Software Suite.

Signed License Keys

Every key is cryptographically signed — it cannot be forged or altered, and it verifies on the user's machine with no internet connection.

Per-Module Access

Choose exactly which Radiant tools each key unlocks — a single module, a handful, or the entire suite.

Trial Periods

Set a trial length in days. Access begins on activation and ends automatically when the period runs out.

Issued-Key Dashboard

Every key you generate is tracked in one place — customer, issue date, expiry, covered modules, and live active or expired status.

Offline Verification

Keys are checked locally inside each application — no license server, no phone-home, no cloud dependency.

Admin-Protected

The console is password-protected and runs only on your own machine — the signing key never leaves it.

How It Works

From key to access.

From the License Manager you generate a key for a customer: enter their name, set a trial length in days, and tick the modules they should receive. The console signs the key and adds it to your dashboard.

The customer enters that key once, into any Radiant tool. From then on, every application re-checks the key each time it launches — unlocking only the licensed modules, and only until the trial expires. No accounts, no servers, nothing to maintain.

From key to access

  • Issue a key — customer, days, modules
  • Send the key to the user
  • They activate any Radiant tool once
  • Access ends when the trial expires

Licensing, handled.

Every application in the Radiant Software Suite ships ready to license. Get in touch to set up a trial of the suite.
