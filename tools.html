Tools

The Quik suite.

A growing collection of internal tools built to solve recurring problems in clinical physics. Each one started as a real workflow bottleneck and turned into something worth keeping.

ESAPI / C#

Radiant QuikQA

Pre-treatment chart-check hub for Eclipse, combining physics verification and contour sanity checks against QUANTEC and TG-101 constraints.

Physics Toolkit

Radiant QuikCalc

Radiobiology calculation suite — BED, EQD2, reirradiation tracking with the Orton & Ellis NSD decay factor, and constraint look-ups.

iOS / SwiftUI

Radiant QuikXpense

Mobile expense tracker for consulting physicists — mileage logs, receipt capture, and year-end exports for Schedule C preparation.

DICOM Utility

DicomSecureSend

Anonymization and secure-transfer tool for DICOM datasets, with audit logging and configurable de-identification profiles.

Flask / Web

CARE

Clinical analysis and reporting environment — a lightweight web app for departmental workflow and reporting tasks.

Custom builds

Radiant builds tools to fit clinical workflows that off-the-shelf software doesn't address. If you have a recurring task that wastes physicist time — an Excel template that should be a script, a chart check that should be automated, a vendor tool that doesn't quite fit — it's likely the right shape of problem for a custom build.
