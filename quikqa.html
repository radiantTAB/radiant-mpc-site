Radiant QuikQA v1.1.2

PyLinac-powered Windows QA tool for radiation oncology physics: Winston-Lutz, Picket Fence, VMAT, TG-142 MLC leaf-speed, TG-51 photon & electron dosimetry, DICOM anonymizer, and PDF tools. Standalone single-file .exe — no install required.

⚠ Evaluation / testing use only — not a medical device

Radiant QuikQA is an unvalidated analysis aid. What's in it

  • Winston-Lutz — MV / kV / CBCT isocenter and BB coincidence.
  • Picket Fence — MLC leaf positional accuracy (TG-142 monthly).
  • VMAT — DRGS and DRMLC MLC speed / dose-rate constancy.
  • TG-142 MLC leaf-speed (new in 1.1.0) — log-based, no portal dosimetry required. Per-leaf RMS error, max single-leaf error, error-at-speed, and beam holds (graded). Achieved leaf speed reported as informational alongside the nominal reference (since 1.1.1). Batch mode now produces a combined PDF (summary on page 1, full per-delivery analyses appended).
  • TG-51 dosimetry — photon and electron reference dosimetry with full chamber and electrometer catalogs.
  • DICOM anonymizer — tag stripping for safely sharing test data.
  • PDF tools — merge, split, rotate, watermark, page numbering, cover.

System requirements

  • Windows 10 / 11, 64-bit
  • ~250 MB free disk for the PyInstaller first-launch unpack cache
  • No installer — double-click RadiantQuikQA.exe

SmartScreen & antivirus

The download is unsigned. Windows SmartScreen may warn “Windows protected your PC” on first launch — click More info → Run anyway if you trust the source. PyInstaller single-file executables occasionally trigger antivirus false-positives because of how they self-unpack at startup; this is a known PyInstaller pattern, not behavior in the application.

Version note

The bundled manual is V 1.0.1; the binary is v1.1.2. The 1.0.1 manual remains accurate for every other module — the additions in 1.1.0 (TG-142 MLC leaf-speed module), the 1.1.1 refinements (achieved leaf speed demoted to informational; richer batch report), and the 1.1.2 UX improvements (taller Setup → Clinics / Machines dialogs; sortable multi-select batch file picker for Leaf-Speed, VMAT Log, and the DICOM Anonymizer) are not yet documented in the PDF. A refreshed manual is in progress.
